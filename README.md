# ConsoleChess

Simple chess implementation for console. UI diplays a board, list of last moves and a text box for in-/output.
Supported Actions:
  - creating new games from a FEN string
  - playing moves (uci)
  - undo moves
  - get legal move suggestions
  - saving to file
  - resizing
